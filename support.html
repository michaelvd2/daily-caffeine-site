Support

Help for the macOS menu bar app.

Basic Use

Daily Caffeine lives in the macOS menu bar. Open the menu to pause, keep awake, add time to today, or open settings.

Schedule

Use settings to choose active weekdays, start time, end time, and an optional break. The app keeps your Mac awake only during the active schedule unless a manual override or Workspace Activity is active.

Add For Today

Add for today extends the current day relative to the scheduled end time. For example, adding 1 hour to an 18:00 end keeps the session active until 19:00 for that day only.

Workspace Activity

Workspace Activity keeps the Mac awake when a selected app is active and files change inside a selected folder. If the same work continues, Daily Caffeine keeps the session going without repeating the same notification.

Manual Pause

Manual pause overrides schedule and Workspace Activity. Use it when you want the Mac to be allowed to sleep even if a schedule or selected coding app would otherwise keep it awake.

Contact

For now, support is handled through the project owner. If you received access to the private release, use the same GitHub or direct contact channel that provided access.
