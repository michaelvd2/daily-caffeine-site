Workspace Activity

Keep late work covered without changing your schedule.

Choose the apps and project folders that count as active work. When a selected app is frontmost and files change inside a selected workspace, Daily Caffeine keeps the Mac awake after scheduled hours.

Notifications stay clear: the app tells you what started the session, then avoids repeating the same workspace update while the same work continues.

01

Daily schedule

Run on selected weekdays with a same-day start and end time. Breaks can pause the assertion during lunch or another fixed window.

02

Manual pause wins

A manual pause overrides schedule and workspace activity, so you can deliberately let the Mac sleep without fighting automation.

03

Add time to today

Add 30 minutes through 8 hours, or an inline custom value, to today's scheduled end. The recurring schedule stays unchanged.

04

Transparent notifications

Session notifications include what started the keep-awake state: schedule, manual override, or workspace activity with app and folder context.

Private by design

No account, no analytics, no remote service.

Daily Caffeine stores its settings locally and uses macOS power assertions to prevent idle sleep. Workspace Activity watches only folders the user selects.

The current build is a private GitHub release for testing. It is not presented here as a Mac App Store approved or notarized public release.
